Measure of Effectiveness Grid Stats Output
------------------------------------------

.. parsed-literal::

   output grid_stats
      file_ <file-name>
      width_ <integer>
      precision_ <integer>
      format_ ...
      justify_ ...
   end_output

Grid stats output will produce a file containing the minimum value, the
maximum value, and the mean value of the measure over the coverage grid.
